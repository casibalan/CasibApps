# CasibApps

CasibApps is an AI-powered USDC invoicing and checkout platform for small businesses and freelancers. The MVP helps merchants create invoices, share payment links, accept USDC payments, and settle transactions on Arc using Circle-powered infrastructure.

## Tech Stack

- Next.js App Router
- React
- TypeScript
- Tailwind CSS

## MVP Roadmap

- AI-assisted invoice creation for merchant payment requests
- USDC checkout links for customer payments
- Circle Developer Platform integration planning
- Arc settlement status and transaction tracking
- Merchant dashboard for invoices, payments, and settlement records

## Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.
